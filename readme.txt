1.1.3 项目开发环境
	Visual Studio Code + Chrome
1.1.4系统功能设定
	一共分为3个主要模块。
	1.TREE:	显示菜单，分为一级和二级菜单；通过对菜单的点击事件选择CENTER的内容
	2.CENTER: 陈列书本；通过拖拽书本加入购物车操作RIGHT的内容
	3.RIGHT: 购物车清单；选择书本购买数量，计算所有书本价格1.1.5 项目开发技术
	HTML+CSS +JQUERY
